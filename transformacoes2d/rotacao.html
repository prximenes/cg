Rotação 2D — Matriz de Rotação

Use as teclas Q/E ou os botões para rotacionar o polígono

Matriz de Rotação R(θ) — coordenadas homogêneas 3×3

R(θ) = | cos θ −sin θ 0 |
| sin θ cos θ 0 |
| 0 0 1 |

A rotação mantem o ponto na mesma distância da origem (preserva distância). Para rotacionar em torno de ponto arbitrário (px, py): T(px,py) · R(θ) · T(−px,−py)

Teclado: Q Girar ←   E Girar →   R Resetar
