Composição: Scale → Rotate → Translate (leitura de baixo para cima)

// Composição: M = T · R · S
// Leitura de baixo para cima:
ctx.translate(200, -30);     // 3a op: translação
ctx.rotate(Math.PI / 6);     // 2a op: rotação 30°
ctx.scale(1.5, 1.5);         // 1a op: escala
desenhaCasa();
