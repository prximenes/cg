Controle independentemente a escala em X e Y com os sliders

Matriz de Escala S(sx, sy) — coordenadas homogêneas 3×3

S(sx, sy) = | sx 0 0 |
| 0 sy 0 |
| 0 0 1 |

Quando sx = sy: escala uniforme (preserva forma).
Quando sx ≠ sy: escala não-uniforme (distorce).
Quando sx = -1 ou sy = -1: reflexão (espelhamento).
