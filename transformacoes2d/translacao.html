Translação 2D — Coordenadas Homogêneas

Use as setas do teclado ou os botões para transladar o polígono

Matriz de Translação (coordenadas homogêneas 3×3)

A translação NÃO pode ser expressa como matriz 2×2 — exige a dimensão extra (homogênea):


T(tx, ty) = | 1 0 tx |
| 0 1 ty |
| 0 0 1 |

Ponto original P = [x, y, 1] → Ponto transformado P' = T · P = [x+tx, y+ty, 1]

No Three.js: mesh.position.set(tx, ty, 0) ou mesh.translateX(tx) atualiza automaticamente a matriz interna do objeto.

Teclado: ← → Move X   ↑ ↓ Move Y   R Resetar
position: (0, 0) | tx acumulado: 0 | ty acumulado: 0
