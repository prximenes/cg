A simetria de 8 octantes

Um círculo é perfeitamente simétrico: calculamos apenas 1/8 dos pontos e refletimos para obter os outros 7 octantes gratuitamente.

p₀ = 1 − r
se p < 0: p = p + 2x + 3
se p ≥ 0: p = p + 2(x − y) + 5, y−−

Para cada ponto (x, y) calculado, pintamos 8 pixels simétricos de uma só vez.

Octante 1: (cx+x, cy−y)
Octante 2: (cx+y, cy−x)
Octante 3: (cx+y, cy+x)
Octante 4: (cx+x, cy+y)
Octante 5: (cx−x, cy+y)
Octante 6: (cx−y, cy+x)
Octante 7: (cx−y, cy−x)
Octante 8: (cx−x, cy−y)
