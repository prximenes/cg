Janela → Viewport — Transformação de Viewing 2D

Mapeamento de coordenadas do mundo para coordenadas de tela

Fórmula de mapeamento Janela → Viewport

xv = vx_min + (xw − wx_min) × (vx_max − vx_min) / (wx_max − wx_min)
yv = vy_min + (yw − wy_min) × (vy_max − vy_min) / (wy_max − wy_min)

Esta é uma interpolação linear que mapeia o intervalo [wx_min, wx_max] para [vx_min, vx_max]. Equivale à normalização do ponto na janela multiplicada pelo tamanho do viewport.
sx = (vx_max - vx_min) / (wx_max - wx_min) é o fator de escala. Se a janela e o viewport tiverem proporções diferentes, haverá distorção.
